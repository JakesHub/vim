# vim  

- `:q` quit  
- `:q!`  quit and discard changes  
- `:wq`
- movement `h` left, `j` down, `k` up, `l` right  
- `x` delete character  
- `dd` delete line  
- `u` undo  
- `:set number`  
- `:2` go to line  
- `i` insert mode  
- `esc` exit to normal mode  
- `+p` paste from system clipboard  
- `:w` save  
- `:!` run command
